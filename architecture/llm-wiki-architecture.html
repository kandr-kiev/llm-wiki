Local LLM Wiki — Architecture

Three-layer knowledge base: Raw Sources → Synthesized Wiki → Schema & Operations

Layer 1 — Raw Sources

  • • Immutable external sources (URL, PDF, paste)
  • • SHA256 hash stored on ingest
  • • Subfolders: articles, papers, transcripts, assets
  • • Source of truth — never modified after ingest
  • • Cron detects new sources every 30 minutes

Layer 2 — Wiki Pages

  • • Mutable, cross-linked knowledge base
  • • 8 subfolders: concepts, entities, comparisons,
  • playbooks, synthesis, queries, references, templates
  • • Hermes Agent orchestrates synthesis
  • • Inter-page wikilinks enable navigation

Layer 3 — Schema & Operations

  • • SCHEMA.md — structural contract (types, tags)
  • • CLAUDE.md — agent operational contract
  • • ALGORITHM.md — step-by-step workflows
  • • index.md — navigation catalog
  • • log.md — append-only audit trail
  • • MCP Server — future integration layer
