<li>Better cost control by letting users choose the right model and compute environment for each task</li>
</ul>

<p class="text-foreground/90">Bionic supports a wide range of coding needs without giving up privacy and control.</p>
<div class="mt-10 flex w-full flex-col items-center justify-center rounded-md text-center"><div class="flex h-full w-full flex-row items-center justify-center"><img src="/assets/marketing/blog/introducing-lm-studio-bionic/Bionic-Coding-workspace-blogimage.png" data-caption="Bionic can inspect local codebases, explain unfamiliar code, and help you make changes." style="width:100%" class="mx-auto mb-3 block h-auto max-w-full rounded-md" alt="Bionic working in a Git codebase"/></div><div class="block w-full rounded-b bg-background/10 pb-5 text-center text-xs opacity-70 hover:opacity-100"><div class="markdown-body blog-markdown-body text-xs"><p class="text-foreground/90 text-xs">Bionic can inspect local codebases, explain unfamiliar code, and help you make changes.</p></div></div></div>
<p class="text-foreground/90">Create a Code project and point it to a local folder. Ask Bionic to investigate, edit, or debug, and review its work as it goes. Inline diffs make every code change easy to inspect, and with agentic code search, Bionic can quickly find relevant files, trace behavior, and explain unfamiliar code.</p>
<p class="text-foreground/90">Bionic works with powerful open models like GLM 5.2 and Kimi K2.7 Code, so you can build more while keeping costs under control.</p>

<p class="text-foreground/90">Use Bionic across documents, PDFs, decks, spreadsheets, and more. In a Work project, Bionic processes documents in a sandboxed environment, keeping the rest of your computer and files safe. It can organize local directories, edit files, summarize materials, and bring outside context into your workflow with native web search. Automatic checkpoints let you safely review or roll back changes, while in-app previews keep your materials and workflow in one place. We&#x27;re continuing to add preview support for more file types, so stay tuned!</p>
