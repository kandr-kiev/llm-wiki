## 1. CORE LOOP ТА ПРОГРЕСІЯ

### 1.1 Стан на сьогодні

| Параметр | Поточна модель |
|----------|----------------|
| **Старт** | 76/8/8/8 (Серце/Кішка/Голова/Трактор) |
| **Ціль** | 25/25/25/25 (Просвітлення) |
| **Джерело прогресії** | Цикл: Підготовка → Вхід → Розвиток → Криза → Фінал → Бардо |
| **Ресурс** | 1 єдина «Енергія» |
| **Тривалість циклу** | 50–120 хв |
| **Проектна кількість циклів** | 100–1000 до першого Просвітлення |
| **Технології** | Unity, Go/Scala/Elixir, Postgres/Redis/Kafka |

### 1.2 Проблеми балансу 76/8/8/8 → 25/25/25/25

**Критична знахідка:** ❌ 25/25/25/25 **математично недосяжний** при поточній системі рішень.

- Домінантна риса (Серце +2) одночасно зменшує суму всіх інших (-0.67 кожна).
- Немає механіки, яка **зменшує домінуючу рису**.
- Гравець може тільки збільшувати обрану рису, тому фатальний дисбаланс зростає, а не стихає.

**Наслідки:**
1. Гравець з початку замкнений на одній рисі (ять монополії).
2. Ілюзія прогресу: гравець рухається вгору по лінії, але не до рівності — до виправданого «краху».
3. Канал зростання занадто вузький: всі рішення обмежені однією силою.

### 1.3 Економіка енергії — дизайн без запасів

| Аспект | Діагноз |
|--------|---------|
| **Інфляція** | Відсутня механіка деградації енергії = накопичення без контексту |
| **Відчуття втрати** | Енергія «зникає» при трансформації — біль від втрати перевершує радість отримання (Loss Aversion) |
| **Вузеньке горло** | Один ресурс на всі операції = жодного вибору стратегії |
| **Залежність від циклу** | Енергія = нагорода за цикл = примусовий геймплейний обов'язок |

**Психологічний ефект:**  
```
Отримав (дофамін) → Витратив (кортизол) → Відчуття «системи з'їдає мої перемоги» → Аверсія до циклу
```

### 1.4 Цикл рішень — agency на межі нуля

| Фаза | Agency | Різноманітність |
|------|--------|-----------------|
| Підготовка | Низька | Вибір фокусу риси — обмежений домінантною рисою |
| Вхід | Мінімальна | Бінарний вхід/відмова |
| Розвиток | Середня, але одностороння | Дії в межах однієї риси — декілька варіантів |
| Криза | Високий, але **реактивний** | Force-major: гравець відповідає, а не вибирає |
| Фінал | Єдина стратегічна точка | Розподіл енергії — останній наділ волі |
| Бардо | **Нульова** | Пасивне спостереження анімації |

**Висновок:** Цикл має **1 сенсову точку agency** — Фінал. Усе інше — анімація або реакція.

## 3. ПСИХОЛОГІЯ ГРАВЦЯ

### 3.1 Джерела монотонності

| Джерело | Чому це втомлює |
|---------|----------------|
| **Жорстка домінанта** | 76% дій — це «діяти з Серцем» без вибору інакшої стратегії |
| **Лінійна прогресія** | Гравець бачить ціль з першої хвилини, але не баче шляху до неї |
| **Пасивний Бардо** | 5 фаз = 20% часу гри = 0 agency = «дивися і чекай» |
| **Один ресурс** | Всі рішення = математика одного числа = жодного вибору стратегії |

### 3.2 Фрустрація від втрати енергії

```
Отримав енергію (дофамін) → Витратив на трансформацію (кортизол) →
«Мої перемоги з'їдає система» → Аверсія до циклу
```

**Три основні болі:**
1. **Втрата = прогрес** — щоб стати кращим, треба втратити ресурс (антагоністично геймплею)
2. **Немає інвестицій** — енергія не приносить % доходу, списується «в пустоту»
3. **Криза = штраф** — фаза, що штовхає гравця до дефіциту

### 3.3 Мотиватори (за Self-Determination Theory)

| Потреба | Стан | Оцінка |
|---------|------|--------|
| **Майстерність** | Рішення тривіальні, немає рівнів майстерності | ❌ Низько |
| **Автономія** | 1 стратегічна точка/цикл | ⚠️ Середньо |
| **Мета** | Філософський каркас (високо) — але роздроблена | ✅ Висока потенція |
| **Соціальність** | Механіки в папері, не в коді | ⚠️ Середньо |
| **Прогрес** | Математичний, не емоційний | ⚠️ Середньо |

## 4. ПРОПОЗИЦІЇ АЛЬТЕРНАТИВ

### 4.1 Динамічний баланс: замість 76/8/8/8 → 25/25/25/25

#### Варіант А: Адаптивний старт
```
Діагностичний квест (3-5 хв) → вибір архетипу → старт 40/30/20/10 (а не 76/8/8/8)
```
**Переваги:** реплейбельність, відчуття унікальності, м'якіший вхід.

#### Варіант Б: Гомеостатична ціль (Homeostatic Target)
```
Ціль НЕ є фіксованим 25/25/25/25.
Ціль = поточний профіль + вектор балансу.
Кожен цикл зсуває ціль на 5-10% до центру.
«Просвітлення» = вектор < 5%.
```
**Переваги:** гравець *налаштовується*, а не *знущується* над собою.

#### Варіант В: Асиметрична спеціалізація
```
Просвітлення = не рівність, а інтеграція в парах:
  40/40/10/10 (Серце+Трактор) або 40/40/10/10 (Кішка+Голова)
```
**Переваги:** зберігає унікальність, дає різні білди.

### 4.2 Як уникнути «Grind»

| Проблема | Рішення | Принцип |
|----------|---------|---------|
| 100+ циклів лінійної математики | **Логарифмічна крива:** перші зміни дешеві, останні = унікальні перки | Прискорення на старті, глибина в кінці |
| Пасивний Бардо | **Активний Бардо:** гравець керує Тінню, проходить 5 міні-ігор з вибором | Agency замість анімації |
| Один ресурс | **4 пули енергії** + конвертер з ентропією | Стратегічний вибір: куди інвестувати |
| Відсутність мікромет | **Іскри/Спаркі:** колекційні фрагменти за нестандартні дії | Винагорода за креативність, не за grind |

### 4.3 Покращення Бардо — «Алхімічна Піч» (5 активних фаз)

| Фаза | Елемент | Механіка | Роль гравця | Продукт |
|------|---------|----------|-------------|---------|
| **Земля** | 🪨 Стійкість | Розміщення «каменів-рішень» на сітці 5×5 | Архітектор | Фундамент наступного циклу |
| **Вода** | 💧 Плинність | Балансування потоків енергії між 4 стихіями | Алхімік | Розчинення дисонансу |
| **Вогонь** | 🔥 Кальцинація | Mini-game: згоряння Тіней (timing/precision) | Воїн | Очищення негативних паттернів |
| **Повітря** | 💨 Сублімація | Вибір 1 єдиної «Ессенції» — перка, що переходить | Мудрець | Наслідок для наступного циклу |
| **Світло** | ✨ Коагуляція | Композиція: збереження Ессенції як артефакту | Творець | Візуалізація Мандали |

> Бардо стає **міні-грою з agency**, де результат впливає на наступний цикл, а не анімація на паузі.

### 4.4 Покращення енергетичної економіки

**Замість 1 Енергії → 4 Пули:**
```
  🔥 Вітальна   — дії, фізичні випробування
  💧 Інтуїтивна — творчість, соціальне
  💨 Ментальна  — аналіз, планування
  ✨ Духовна    — трансформація, зцілення
```

**Конвертер з ентропією:**
```
Втрачаєш 2 одиниці однієї → отримуєш 1 іншої.
Ентропія: 10% втрати на кожну конверсію.
Очищення: фаза Вогонь у Бардо.
```

**Ефект:** економіка стає **багатовимірною**, вибір стратегії обов'язковий.

## 5. АЛЬТЕРНАТИВНА СИСТЕМА 4-Х РИС

### 5.1 Нова номенклатура (зберігає кабалістичну основу, посилює семантику)

| Стара риса | Нова риса | Архетип | Дієслово | Даосізм | Буддизм |
|------------|-----------|---------|----------|---------|---------|
| ❤ Серце | **💠 Компасія** | Цілющий | *Зцілювати/З'єднувати* | Земля | Каруна |
| 🐱 Кішка | **🌊 Інтуїція** | Танцюрист | *Відчувати/Плисти* | Вода | Праджня |
| 🧠 Голова | **⚡ Ясність** | Стратег | *Пізнавати/Розрізняти* | Вогонь | Саммаді |
| 🚜 Трактор | **🏔 Воля** | Будівельник | *Творити/Утримувати* | Метал | Вірія |

### 5.2 Синергії (6 комбінацій + активний стан при ≥30/30)

| Комбінація | Архетип | Здатність | Ефект |
|-------------|---------|-----------|-------|
| 💠 + 🌊 | **Муза** | Реонанс | Подвоєна соціальна енергія за цикл |
| 💠 + ⚡ | **Суддя** | Вірдикт | Перегляд Кризи — виправляє 1 помилку |
| 💠 + 🏔 | **Опікун** | Бастіон | +1 захист на наступний цикл |
| 🌊 + ⚡ | **Відьмак** | Провідіння | Підказка майбутньої Кризи |
| 🌊 + 🏔 | **Потік** | Ву-вей | Автоматичне балансування стихій |
| ⚡ + 🏔 | **Інженер** | Система | Автоматизація однієї рутини |

**Умова активації:** Primary_A ≥ 30 AND Primary_B ≥ 30.

### 5.3 Система Тіней (Shadow / Нафс)

**Активація:** автоматична при Primary > 60.

| Тінь | Тип | Ефект | Інтеграція → Архетип |
|------|-----|-------|---------------------|
| 💔 Тиранія Компасії | Пасивна | -30% чистоти сердечних рішень | → 🕊️ Мірний Воїн |
| 😈 Обман Інтуїції | Активна | -20% достовірності підказок | → 🎭 Маестро Масок |
| 🧊 Холод Ясності | Пасивна | Раціоналізація жорстокості | → ❄️ Аналітик Хладу |
| 🪨 Тупість Волі | Активна | Жорсткість, рутинізація | → 🗿 Нерухомий |

**Інтеграція Тіні:** витрата енергії + проходження Бардо-фази → Тінь → Архетип Тіні (унікальна пасивна здатність).

### 5.4 Графова прогресія замість лінійної

```
START (архетипний профіль 40/30/20/10)
   │
   ├── СПЕЦІАЛІЗАЦІЯ (40/40/10/10) → Синергія
   │         │
   │         ├── БАЛАНС (25/25/25/25) + Всі 4 Синергії = Просвітлення
   │         │
   │         └── ПЕРЕКОС (60/20/10/10) → Тінь → Інтеграція → Унікальний Архетип
   │
   └── ДРУГИЙ ШЛЯХ...
```

Кожен вузол = унікальний сет здатностей і варіацій Бардо. Гравець **обирає напрямок**.

### 5.5 Математика ваг (модель)

```
Перші 10 одиниць: ×0.5 (легко розвивати слабке)
10 → 30: ×1.0
30 → 40 (поріг синергії): ×1.5
40 → 50: ×2.0
50 → 60 (поріг тіні): ×3.0
>60: ×5.0 + ризик Тіні
```

| **Синергії** | Відсутні | 6 активних (≥30/30) + 4 тіньові |
| **Мультиплеєр** | PvP у Пустці | Асоціативний: спільні квести, резонанс-матчинг |
| **Технології** | 6-мовний моноліт | Go + Postgres + Redis для MVP |

## 9. ВИСНОВКИ

### 🔴 Критичні прогалини (Must Fix)
1. **Математична недосяжність 25/25/25/25** — система не має механіки зменшення домінуючої риси.
2. **Єдина енергія** — стирає філософську різницю між традиціями, створює одномірність.
3. **Пасивний Бардо** — 20% часу гри з 0 agency = вертикальна втрата залучення.
4. **Grind 100+ циклів** — четверта частина життя гравця = виснаження, а не просвітлення.
5. **Енергія = покарання** — гравець платить за прогрес, а не отримує його.

### 🟠 Високі пріоритети
6. **Трактор/Кіт** — когнітивний дисонанс; кіт не = Хесед, трактор не = Гевура.
7. **Поверхневий мультиплеєр** — Ехо/Резонанс/Синтез задекларовані, не прототиповані.
8. **Баланс = рівність** — 25/25/25/25 — не просвітлення, а усереднення.

### 🟢 Швидкі виграші (Quick Wins)
1. Додати стрілку «вектора балансу» на UI.
2. Розділити Енергію на 4 пули візуально.
3. Додати 1 міні-вибір у кожну фазу Бардо.
4. Спарки/Іскри — колекційні фрагменти лору.
5. Синергії, прив'язані до Ехо/Резонансу.

### 📐 Рекомендація
**Гра не може вийти в поточному вигляді.** Її ядро має математичні та психологічні неполадки, які призводять до відторгнення гравців. Перший крок — виправити пріоритети (поточна MVP надмірно розширена, але надто вузенька в глибині). Рекомендується створити прототип **Core 4** (Баланс + Синергії + Тіні + Активний Бардо) і провести плейтести на 30-45 хвилинних сесіях перед розширенням мультиплеєра.
