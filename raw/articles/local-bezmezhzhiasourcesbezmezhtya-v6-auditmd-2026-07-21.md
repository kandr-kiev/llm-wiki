## 1. CORE LOOP ТА ПРОГРЕСІЯ

### 1.1 Аналіз початкового балансу (76/8/8/8)

| Риса | Початок | Фінал | Дельта | % від загального |
|------|---------|-------|--------|------------------|
| ❤ Серце (Тіферет) | 76 | 25 | -51 | 76% → 25% |
| 🐱 Кішка (Хесед) | 8 | 25 | +17 | 8% → 25% |
| 🧠 Голова (Хохма) | 8 | 25 | +17 | 8% → 25% |
| 🚜 Трактор (Гевура) | 8 | 25 | +17 | 8% → 25% |
| **СУМА** | **100** | **100** | — | — |

**Критичні спостереження:**

1. **Екстремальний розкид (76 vs 8)** — початковий стан створює *ілюзію вибору*, а не реальний вибір. Гравець починає з «переважанням Серця», але механічно це означає: 76% дій доступні лише через однієї риси.
2. **Жорстка симетрія фіналу (25/25/25/25)** — вимагає від гравця *активно знижувати* домінантну рису. Це контрінтуїтивно: більшість RPG нагороджують спеціалізацію, а тут — змушують «розмазуватися».
3. **Математика трансформації:** За 3 цикли (прибл.) гравець має перерозподілити 51 одиницю. Це ~17 одиниць на цикл = **grind по суті**.

### 1.2 Економіка енергії

```
Цикл: Підготовка → Вхід → Розвиток → Криза → Фінал → Бардо
                          ↓
                    Нагорода: Енергія
                          ↓
              Витрата на Трансформацію (зміна рис)
```

**Проблеми:**
- **Жодного інструмента覧ю економіки:** немає візуалізації «вартості» перерозподілу.
- **Енергія = єдиний ресурс** — створює вузьке горло. Всі рішення зводяться до «чи вистачає енергії».
- **Втрата енергії = фрустрація** (див. розділ 3). Гравець *платить* за прогрес, а не *отримує* його.
- **Немає «процентного» витрачання** — лише абсолютні числа. При низькій енергії рішення стають боліснішими.

### 1.3 Цикл рішень (Decision Loop)

| Фаза | Тип рішення | Різноманітність | Ризик |
|------|-------------|-----------------|-------|
| Підготовка | Вибір фокусу риси | Низька (dominaтна риса диктує) | Низький |
| Вхід | Бінарний (вхід/відмова) | Мінімальна | Низький |
| Розвиток | Вибір дій у межах риси | Середня (але обмежена однією рисою) | Середній |
| Криза | Force-major (RNG/скрипт) | Низька (гравець реактивний) | Високий |
| Фінал | Розподіл енергії | **Єдина стратегічна точка** | Критичний |
| Бардо | Пасивне спостереження | Нульова | — |

**Висновок:** Цикл має **лише 1 смислову точку agency** — Фінал. Усе інше — програвання анімацій/тексту.

### 2.4 Пропозиція: Різниця енергій за традиціями

```
Замість 1 Енергії → 4 Пули енергії:
  🔥 Вітальна (Гевура/Трактор) — витрачається на дії, фізичні випробування
  💧 Інтуїтивна (Хесед/Кішка) — витрачається на творчість, соціальне
  💨 Ментальна (Хохма/Голова) — витрачається на аналіз, планування
  ✨ Духовна (Тіферет/Серце) — витрачається на трансформацію, зцілення
```

Це робить економіку **багатовимірною** і змушує гравця балансувати *внутрішньо*, а не лише математично.

## 3. ПСИХОЛОГІЯ ГРАВЦЯ

### 3.1 Монотонність: джерела

| Джерело | Механіка | Чому це монотонно |
|---------|----------|-------------------|
| **Жорстка домінанта** | 76% Серця на старті | 76% дій однакових — «діяти з серцем» |
| **Лінійна прогресія** | 76→25 по Серцю, 8→25 по іншим | Гравець *знає* кінцеву точку з хвилини 1 |
| **Пасивний Бардо** | 5 фаз без agency | «Дивися анімацію» × N циклів |
| **Єдиний ресурс** | Енергія на все | Всі рішення = математика одного числа |

### 3.2 Фрустрація від «втрати енергії»

**Психологічний профіль:**
```
Отримав енергію (дофамін) → Витратив на трансформацію (кортизол) → 
Ефект: «Мої нагороди з'їдає система» → Аверсія до циклу
```

**Ключові болі:**
1. **Втрата = прогрес** — парадокс: щоб стати кращим, треба *втратити* ресурс.
2. **Немає «інвестицій»** — енергія не приносить % доходу, лише списується.
3. **Криза як штраф** — фаза Криза часто відбирає енергію (RNG), подвоюючи фрустрацію.

### 3.3 Мотиватори: що працює / що не працює

| Мотиватор | Поточний стан | Потенціал |
|-----------|---------------|-----------|
| **Майстерність (Mastery)** | Низький — рішення тривіальні | Високий — якщо додати синергії рис |
| **Автономія (Autonomy)** | Низький — 1 стратегічна точка/цикл | Середній — розширити agency у фазах |
| **Мета (Purpose)** | Високий — філософський каркас | Високий — але роздроблений |
| **Соціальність (Relatedness)** | Echo/Resonance/Synthesis — непрограмовані | Високий — якщо прив'язати до механік |
| **Прогрес (Progress)** | Математичний, не емоційний | Середній — візуалізація трансформації |

## 4. ПРОПОЗИЦІЇ АЛЬТЕРНАТИВ

### 4.1 Динамічний баланс (замість 76/8/8/8 → 25/25/25/25)

#### Варіант А: «Адаптивний старт» (Procedural Archetype)
```
На старті — короткий діагностичний квест (3-5 хвилин):
  → Результат: один з 16 архетípів (4^2 комбінації домінант/рецесив)
  → Початкові риси: 40/30/20/10 (а не 76/8/8/8)
  → Кожен архетіп має унікальний стартовий шлях у Бардо
```
**Переваги:** Реплейбельність, відчуття унікальності, м'якший вхід.

#### Варіант Б: «Динамічна рівновага» (Homeostatic Target)
```
Фінальний стан НЕ фіксований 25/25/25/25, а:
  → Ціль = поточний профіль гравця + вектор балансу
  → Кожен цикл зсуває ціль на 5-10% доіцентру
  → «Просвітлення» = коли вектор стає < 5% (динамічна рівновага)
```
**Переваги:** Гравець не «знущується» над собою, а *налагоджується*.

#### Варіант В: «Асиметрична Трансформація»
```
Риси не рівняються — вони *спеціалізуються* в парах:
  → Серце + Кішка = Архетип «Творець» (Empathic Creator)
  → Голова + Трактор = Архетип «Архітектор» (Disciplined Mind)
  → Серце + Трактор = Архетип «Воїн Світла» (Compassionate Force)
  → Кішка + Голова = Архетип «Мудрий Кот» (Intuitive Sage)
Просвітлення = досягнення 40/40/10/10 у *своїй парі* + 10/10 в інших
```

### 4.2 Уникнення Grind

| Проблема | Рішення |
|----------|---------|
| **Лінійна математика** | **Експоненційні/логарифмічні криві:** перші 20% змін — дешево, останні 20% — дорого, але дають унікальні здібності (перки). |
| **Пасивний Бардо** | **Активний Бардо:** гравець керує «Тінню» — проєкцією свого его, яка борється за енергію. Перемога = бонус до наступного циклу. |
| **Єдиний ресурс** | **4 пули енергії** (див. 2.4) + **Конвертер** (витрачає 2 одиниці однієї → 1 іншої, з штрафом ентропії). |
| **Відсутність побічних мет** | **Колекціонування «Спарків» (Іскр):** унікальні фрагменти пам'яті циклів, що відкривають лор/скіни/здібності. Необов'язково, але мотивує ексperimentи. |

### 4.3 Покращення Бардо: з пасивного в активний

**Поточний Бардо:** Земля → Вода → Вогонь → Повітря → Світло (5 скролів/відео)

**Новий Бардо — «Алхімічна печь» (5 активних фаз):**

| Фаза | Метафора | Механіка геймплею | Роль гравця |
|------|----------|-------------------|-------------|
| **Земля** | Коагуляція (Стілл) | Розміщення «камінняв» (рішень циклу) на сітці 5×5. Формування фундаменту. | Архітектор |
| **Вода** | Розчинення (Сольве) | Плинний пазл: спрямування потоків енергії між каміннями. Баланс 4-х стихій. | Алхімік |
| **Вогонь** | Кальцинація | Mini-game: «згоряння» Тіней (негативних патернів циклу). Timing/precision. | Воїн |
| **Повітря** | Сублімація | Вибір «Ессенції» — одного перка/здатності, що переходить у новий цикл. | Мудрець |
| **Світло** | Коагуляція-2 | Фінальна композиція: збереження Ессенції як артефакту. Візуалізація Мандали. | Творець |

**Ключова зміна:** Бардо стає **міні-грою з agency**, де результат впливає на наступний цикл (перки, стартову енергію, доступні архетіпи).

```

  → Негативний ефект: -10% енергії за цикл, спотворення подій Кризи
  → Бардо: додаткова фаза «Зустріч з Тінню» (бій/діалог/інтеграція)
```

### 5.5 Ваги та конверсія (математична модель)

```
BASE_COST = 1 одиниця риси = 1 Енергії (базово)

МОДИФІКАТОРИ ВАРТОСТІ:
  • Перші 10 одиниць з 8 → 18: ×0.5 (легко розвивати слабке)
  • 18 → 30: ×1.0 (нормально)
  • 30 → 40 (поріг синергії): ×1.5 (дорого, але анлок)
  • 40 → 50: ×2.0
  • 50 → 60 (поріг тіні): ×3.0 (небезпечно)
  • >60: ×5.0 + ризик Тіні

КОНВЕРТЕР ЕНЕРГІЇ (у Бардо):
  Втрата 2 Енергії одного типу → 1 Енергії іншого типу
  Ентропія: 10% втрати на кожну конверсію (накопичується)
  Очищення ентропії: фаза Вогонь у Бардо
```

## 6. ПІДСУМКОВІ ВИСНОВКИ ТА РОАДМАП

### 6.1 Критичні прогалини (Must Fix)

| # | Проблема | Приорітет | Оцінка зусиль |
|---|----------|-----------|---------------|
| 1 | **Єдиний ресурс енергії** стирає filozofічну різницю традицій | 🔴 Critical | Висока (рефакторинг економіки) |
| 2 | **Пасивний Бардо** — 0 agency на 20% часу гри | 🔴 Critical | Середня (дизайн 5 міні-ігор) |
| 3 | **76/8/8/8 старт** — ілюзія вибору, жорстка домінанта | 🟠 High | Низька (зміна чисел + архетіпи) |
| 4 | **Втрата енергії = фрустрація** (не інвестиція) | 🟠 High | Середня (перепроектування нагород) |
| 5 | **Трактор/Кіт** — слабкі метафори для Гевури/Хеседу | 🟡 Medium | Низька (перейменування/рефлейвор) |

### 6.2 Швидкі виграші (Quick Wins, 1-2 спринти)

1. **Додати візуалізацію «Вектора балансу»** — стрілка на UI, що показує напрямок до 25/25/25/25.
2. **Розділити Енергію на 4 пули** (візуально, навіть якщо математично однакові).
3. **Бардо: додати 1 вибір на фазу** (навіть косметичний — колір, символ, фраза).
4. **Спарки (Іскри)** — колекціонерські фрагменти лору за нестандартні дії.
5. **Ехо/Резонанс:** прив'язати до синергій (Муза дає бонус до Резонансу).

### 6.3 Стратегічні зміни (Major Version v7.0)

1. **Графове прогресія** замість лінійної (розділ 5.4).
2. **Синергії + Тіні** як повноцінні механічні шари (розділ 5.1-5.3).
3. **Активний Бардо** як міні-гра з перками (розділ 4.3).
4. **Адаптивний старт** через діагностичний квест (розділ 4.1).
5. **Мультиплеєр через синергії:** Синтез = об'єднання синергій двох гравців.

## 7. ТЕХНІЧНІ ПРИМІТКИ (Stack: Unity / Go-Scala-Elixir / PG-Redis-Kafka)

| Компонент | Поточний ризик | Рекомендація |
|-----------|----------------|--------------|
| **Енергія як єдиний int в PG** | Не масштабується до 4-х пул + ентропія | Міграція на `energy_state JSONB` з версіонуванням |
| **Бардо як скриптована послідовність** | Не підтримує agency | Перехід на **Event Sourcing** (Kafka): кожна фаза = подія з payload вибору гравця |
| **Синергії/Тіні** | Нова логіка | Реалізувати в **Elixir** (GenServer на гравця) — природне моделирование станів/переходів |
| **PvP Пустка** | Потенційно токсична | Додати **Резонанс-матчинг**: парить гравців з *комплементарними* синергіями, а не рівнем |
