# translate()
![](https://secure.gravatar.com/avatar/0387aa3c461f7db63d04d88bb5968631b7c3e2a22b00a9a6bd81f352449654d9?s=80&d=retro&r=pg)

The CSS `translate()` function shifts an element from its default position on a two-dimensional plane. This way, we can reposition an element horizontally, vertically, or both.
.parent:hover .box {
transform: translate(50px, 50%);
}
Hover over the box to see it move 50% of its width towards the left:

Along with other transform functions, it is used inside the [transform](https://css-tricks.com/almanac/properties/t/transform/) property.
The `translate()` function is defined in the [CSS Transforms Module Level 1](https://drafts.csswg.org/css-transforms-1/#funcdef-transform-scale) draft.
