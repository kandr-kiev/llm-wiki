Цей робочий процес пояснює, як запустити проект на локальному комп'ютері для розробки.
// turbo
1. Переконайтеся, що у вас встановлено PHP. Перевірте версію:
```powershell
php -v
```
2. Запустіть вбудований PHP-сервер у кореневій директорії проекту:
```powershell
php -S localhost:8000
```
3. Відкрийте браузер за адресою `http://localhost:8000/index2.html`.
4. Для роботи з базою даних переконайтеся, що MySQL запущено (наприклад, через XAMPP або OpenServer) і налаштування в `api.php` відповідають вашим локальним даним.
